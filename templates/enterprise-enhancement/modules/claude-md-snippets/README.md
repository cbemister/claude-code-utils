# CLAUDE.md Snippets

Ready-to-paste sections for your existing CLAUDE.md. Each snippet is self-contained.

## How to Use

1. Open your project's `CLAUDE.md`
2. Choose which snippets apply to your project
3. Paste each snippet in the appropriate location
4. Customize `[PLACEHOLDER]` values

## Snippet Placement Guide

| Snippet | Place after |
|---|---|
| `agent-team-guide.md` | Key Commands section |
| `rules-guide.md` | Project Structure section |
| `context-management.md` | Development Workflow section |
| `enterprise-standards.md` | Near the end, before any Notes section |

## After Installing

Run through your updated CLAUDE.md and:
1. Replace all `[PLACEHOLDER]` values
2. Remove any sections that don't apply to your project
3. Adjust the agent workflow table to match agents you actually installed
